# Trader Performance vs Market Sentiment

## Objective
Analyze how market sentiment affects trader behavior and performance.

## Methodology
- Data cleaning and preprocessing
- Merging sentiment with trading data
- Feature engineering (PnL, trade count, trade size)
- Comparative analysis across sentiment categories

## Key Insights
1. Higher profitability during Fear periods
2. Increased trading activity during Extreme Fear
3. Larger trade sizes observed during Fear

## Strategy Recommendations
1. Increase participation during Fear periods
2. Apply risk control during Greed periods
